# Projeto 004 — Análise de Vendas de Coffee Shop (Dataset Kaggle)

## Problema
Muitas cafeterias (pequenas ou médias) possuem dados brutos de vendas, mas não têm tempo nem estrutura para transformar isso em insights práticos que ajudem no faturamento diário.
O dataset do Kaggle (fictício, mas realista) simula exatamente esse cenário: milhares de transações desconexas, sem narrativa.

## Solução
Um mini-projeto de análise limpa e objetiva:
Código curto (menos de 80 linhas úteis)

Gráficos simples e interpretáveis
Insights acionáveis que qualquer dono de cafeteria entenderia
Dados 100% reais do dataset Kaggle

## O que o projeto entrega
✔ Tendência de vendas ao longo do tempo
✔ Quais dias da semana são mais fortes
✔ Produtos que mais vendem
✔ Produtos que mais geram dinheiro
✔ Micro-insights que justificam cada achado

# Como rodar
pip install pandas matplotlib
python main.py

# Dataset utilizado
"Coffee Shop Sales" — Kaggle (Maven Roasters).
Dataset totalmente público para fins educacionais.
